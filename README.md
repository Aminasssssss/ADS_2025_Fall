# Algorithms and Data Structures — KBTU Fall 2025

Course: Algorithms and Data Structures
University: Kazakh-British Technical University (KBTU)

## Course Overview

Core computer science concepts covering fundamental data structures and algorithms.
Focus on algorithm analysis, complexity, and practical implementation.

## Topics Covered

- Algorithm complexity: Big O, Big Omega, Big Theta
- Arrays and linked lists
- Stacks and queues
- Trees: binary trees, BST, AVL trees
- Heaps and priority queues
- Hash tables and hash functions
- Graphs: BFS, DFS, shortest paths
- Sorting algorithms: quicksort, mergesort, heapsort
- Searching algorithms
- Dynamic programming
- Greedy algorithms
- Recursion and backtracking

## Tech Stack

- Python
- Jupyter Notebook
